# Meal Planner & Nutrition Tracker

تطبيق لتخطيط الوجبات وتتبع التغذية اليومية.

## المميزات
- اقتراح وجبات بناءً على السعرات المستهدفة
- تسجيل الوجبات المتنوعة
- حساب السعرات والعناصر الغذائية (بروتين، كربوهيدرات، دهون)
- واجهة مستخدم بسيطة باستخدام Streamlit

## كيفية التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

افتح المتصفح على http://localhost:8501

## المستقبل
- دمج قاعدة بيانات حقيقية (API للوصفات)
- توصيات مخصصة بناءً على التفضيلات الصحية
- تقارير أسبوعية
